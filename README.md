# rhamburg.com

| Command           | Action                                           |
| ----------------- | ------------------------------------------------ |
| `pnpm dev`    	| Starts local dev server at `localhost:4321`      |
| `pnpm build` 		| Build your production site to `./vercel/`        |
| `pnpm preview`	| Preview your build locally, before deploying     |
| `pnpm clean`	    | Remove build output directories                  |
